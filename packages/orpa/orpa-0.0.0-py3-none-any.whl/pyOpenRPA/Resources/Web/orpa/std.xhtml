Режим поиска UI объекта по курсору мыши

!ВНИМАНИЕ! НАЧИНАЯ С ВЕРСИИ ORPA 1.4.1 ПОМЕНЯЛИСЬ ФУНКЦИИ ГОРЯЧИХ КЛАВИШ, ОПИСАНИЕ СМ. НИЖЕ!

Уважаемый пользователь!

Инициирован режим поиска локального UI объекта по наведению курсора мыши.

Через несколько секунд ORPA перенесет вас в приложение для поиска объекта.

Ниже представлены доступные команды:

  • Длительное нажатие shift (левый или правый): Завершить поиск и вернуть UI объект, который был обнаружен по наведенному курсору мыши
  • Длительное нажатие alt (левый или правый): Вывод всех Ui объектов, которые удовлетворяют условию нахождения под координатами мыши. Данная опция позволяет найти Ui объект, даже в том случае, если он перекрыт другим Ui слоем.
  • Длительное нажатие ctrl (левый или правый): Фиксация координат мыши на момент начала нажатия клавиши shift. Данная опция позволяет идентифицировать UI объект в том случае, когда он активен без наведенного курсора мыши.

Данное модальное окно будет открыто до момента завершения режима поиска.
